# 极简阅读器

> 一个无需联网、纯本地运行、智能生成目录的 Web 版 TXT 阅读器。

![License](https://img.shields.io/badge/license-MIT-green) ![HTML5](https://img.shields.io/badge/language-HTML5-orange) ![Status](https://img.shields.io/badge/status-stable-blue)

## 📖 简介

**极简阅读器** 是一款基于 HTML5 开发的单文件电子书阅读器。它专为深度阅读设计，旨在解决 TXT 文档在电脑端阅读体验差、缺乏目录导航、排版混乱等痛点。

它完全运行在浏览器端，**不上传任何文件到服务器**，确保你的隐私安全。

## ✨ 核心功能

* **🛡️ 隐私安全**：基于 `FileReader` API，所有书籍解析均在本地浏览器内存中完成。
* **📑 智能目录**：自动识别文档中的“第X章/节/卷”格式，生成左侧可交互目录（支持点击跳转）。
* **🎨 四款主题**：
    * **纯净白**：极简风格。
    * **护眼绿**：经典的豆沙绿，长时间阅读不疲劳。
    * **羊皮纸**：复古暖色调。
    * **夜间模式**：深色背景，适合暗光环境。
* **👁️ 沉浸体验**：
    * **目录联动**：阅读进度与目录实时同步，自动高亮当前章节。
    * **状态监控**：右下角悬浮显示 `书名 | 章节 | 进度百分比`。
* **🔤 字体优化**：完美兼容宋体、黑体及楷体（自动适配 Windows/Mac 系统字体栈）。

## ⚙️ 常见问题 (FAQ)

### Q: 打开文件后显示全是乱码（菱形问号 ）怎么办？
**A:** 这通常是因为 TXT 文件的编码格式问题。
* 本阅读器默认使用国际通用的 **UTF-8** 编码。
* 如果你的文件是老式的 **GBK** 编码（常见于旧电脑上的小说），请使用记事本将文件“另存为”并选择 UTF-8 编码，或者修改源码中的 `readAsText(file, 'UTF-8')` 为 `readAsText(file, 'GBK')`。

### Q: 为什么没有生成目录？
**A:** 目录生成依赖于正则表达式匹配。目前规则支持识别 `第` + `数字` + `章/节/卷` 的格式。如果书籍章节标题不规范，将只显示正文。

### Q: 手机上能用吗？
**A:** 可以。该页面适配了移动端布局，侧边栏会自动折叠，点击左上角 `☰` 即可呼出目录。
